# 美化与功能改进路线图

记录后续需要做的视觉与体验改进。按改动量分级，优先级自上而下。

---

## 低改动量（半天内）

### 1. 模块色彩化
当前全部模块共用同一蓝色 accent，视觉上彼此难区分。为七个模块各分配独立主题色，仅改 icon 容器背景和标题条辅助色，主体面板保持白玻璃：

| 模块 | Accent |
|------|--------|
| 打卡记录 | 蓝 `#1d4ed8`（保留） |
| 手机克制 | 石板 `#475569` |
| 任务管理 | 琥珀 `#d97706` |
| 专注时长 | 紫 `#7c3aed` |
| 久坐提醒 | 青绿 `#0d9488` |
| 请假记录 | 玫红 `#e11d48` |
| 统计分析 | 靛蓝 `#4338ca` |
| 规则设置 | 石墨 `#334155` |

### 2. 左侧导航状态化
- 激活项加左侧 3px 竖线指示条
- hover 时图标轻微放大 + accent 色填充
- 每项尾部放小徽标：未打卡 ⚠️、进行中任务数、今日专注分钟

### 3. 头部升级为 Dashboard Hero
现在是两行纯文字。重排为：
- 左：大字日期 + 问候语（上午好 / 下午好 / 晚上好）
- 中：KPI pill（今日专注 / 已完成任务 / 打卡进度）
- 右：小号猫咪头像 + 状态

---

## 中改动量（1–2 天）

### 4. 计时器改圆环
专注时长与久坐提醒的大号数字 + 线性进度条，换成 SVG 圆环进度（数字居中，环带渐变描边 + `stroke-dasharray` 动画）。视觉权重最高的区域，升级收益最大。

### 5. 统计分析可视化强化
- Chart.js 柱状图改为渐变填充 + 圆角顶
- 新增 GitHub 风格热力图（近 90 天打卡 + 专注强度叠加）

### 6. 深色模式
`:root` 已经用 CSS 变量，扩一套 `[data-theme="dark"]` 覆盖 5–6 个变量即可；头部加切换按钮。

---

## 高改动量（按需挑选）

### 7. 猫咪拟人化反馈
已有 happy / trusting / adoring 表情。让它响应用户行为：
- 久坐超时 → 困倦 / 无精打采
- 完成专注 → 闪星星 / 鼓掌
- 请假日 → 睡觉
- 深夜工作 → 戴睡帽打哈欠

从装饰升级为陪伴 agent。

### 8. 微动效系统化
- 打卡成功 → 粒子 / confetti
- 完成专注 → 数字 count-up
- 任务打钩 → 划线 + 轻弹

## 已知待修问题

### 跨午夜下班（进行中）
如果 00:00 之后才下班，当前系统：
1. 晚上下班按钮被禁用（超出 20:00–24:00 区间）
2. 即使按下，数据会写入新一天的 `checkinData[today].evening.checkOut`，昨晚记录保持"未下班"

解决方案见 `docs/CROSS_MIDNIGHT_FIX.md`。
